Hello world

Welcome to my personal Mask external plugin!

Mask Network SDK is available with window.Mask

Any API described below is experimental, it will change in the future.

Mask.persona

Interact with Personas in Mask Network.

Mask.persona.__experimental__sign__(message: string, style: "web3")

Sign message with a persona.

Mask.socialNetwork

Interact with the Social Network Service that opens this page.

readonly Mask.socialNetwork.connected: boolean

If this page is connected to some SNS page.

readonly Mask.socialNetwork.metadata: unknown

Payload that this plugin previously appended to a Mask Network post.

Mask.socialNetwork.appendComposition(message: string, metadata?: Map<string, unknown>)

Append message to the connected SNS. If Mask.socialNetwork.connected is false, call this function will return an error.

The message can be read back again when it is opened in SNS with Mask.socialNetwork.metadata.
